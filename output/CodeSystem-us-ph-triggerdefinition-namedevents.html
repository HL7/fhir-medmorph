CodeSystem: US Public Health TriggerDefinition Named Events

Summary

Defining URL:http://hl7.org/fhir/us/fhir-medmorph/CodeSystem/us-ph-triggerdefinition-namedevents
Version:0.1.0
Name:USPublicHealthTriggerDefinitionNamedEvents
Title:US Public Health TriggerDefinition Named Events
Status:Active as of 2020-09-06
Definition:

The US Public Health TriggerDefinition NamedEvents CodeSystem is a 'starter set' of codes supported for identifying named events to subscribe to in clinical workflows.

Publisher:HL7 International - Public Health Work Group
Content:Complete: All the concepts defined by the code system are included in the code system resource
Value Set:http://hl7.org/fhir/us/fhir-medmorph/ValueSet/us-ph-triggerdefinition-namedevent ( is the value set for all codes in this code system)
Source Resource:XML / JSON / Turtle

This Code system is referenced in the content logical definition of the following value sets:

  • USPublicHealthTriggerDefinitionNamedEvent

This code system http://hl7.org/fhir/us/fhir-medmorph/CodeSystem/us-ph-triggerdefinition-namedevents defines the following codes:

LvlCodeDisplayDefinition
